📱 Access From Lock Screen

Method 1: iOS Shortcuts (Recommended)

  1. Open the Shortcuts app on your iPhone.
  2. Tap + to create a new shortcut.
  3. Tap Add Action → search for "Open URLs".
  4. Enter the URL of this page (e.g. your hosted link or local server URL).
  5. Tap the shortcut name → Add to Home Screen.
  6. Go to Settings → Accessibility → Touch → Back Tap.
  7. Set Triple Tap → choose your new shortcut.

Method 2: iOS Accessibility

  1. Go to Settings → Accessibility → Guided Access.
  2. Alternatively: Settings → Accessibility → Shortcuts.
  3. Set the shortcut to open Safari with this page's URL.
  4. Triple-click the side button to trigger it from the lock screen.

Tip: Add to Home Screen

In Safari, tap the Share icon → Add to Home Screen. This creates an icon on your home screen that launches directly to this page — no URL bar.
